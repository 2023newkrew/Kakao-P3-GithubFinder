# Kakao-P3-GithubFinder

<h2>실행 안내</h2>

1. 설치

```
npm install
```

2. 빌드

```
npm run build
```

3. 로컬 서버 구동

```
npm run dev
```

---

<h2>alias 설정 안내</h2>

| 이름     | 경로         |
| -------- | ------------ |
| @        | /src         |
| @scripts | /src/scripts |
| @styles  | /src/styles  |
| @views   | /src/views   |
| @assets  | /src/assets  |
